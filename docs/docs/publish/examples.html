  • The slowest continued fraction converges to the golden ratio. 118 iterations are needed to recover 50 digits of precision:
x = gem(1);
for i = 1:118
    x = 1/(1+x);
    if mod(i-8,10) == 0
        disp(num2str(1+x,50));
    end
end
